电影院管理系统

6个类：

1.Main
2.Venue
3.Theater
4.Seat
5.Movie
6.Showtime

Main类包含Main方法，负责用户界面和交互。它提供了一个包含各种选项的菜单，例如添加和删除电影和放映时间、购买门票以及查看电影和座位信息。Main类还使用
Movie数据库（TMDB）API获取电影详细信息并填充Movie对象。

Theater类代表电影院。它有几个属性，包括Movie对象列表、Showtime对象列表和Seat对象的二维数组。Theater类有几个方法，允许用户添加、删除和查看电影、
放映时间和座位信息。它还有一些方法，允许用户通过标题、时间和日期购买门票并查找电影和放映时间。

Seat类代表电影院的座位。它有几个属性，包括座位的行号和列号，以及一个指示座位是否可用的布尔值。Seat类有几个getter和setter方法，允许用户访问和修改这些属性。

Movie类代表一部电影。它有几个属性，包括电影的标题、上映年份、导演、持续时间和类型。Movie类有几个getter和setter方法，允许用户访问和修改这些属性。

Showtime类表示电影院中电影的放映时间。它有几个属性，包括电影的Movie对象、放映时间和日期以及票价。Showtime类有几个getter和setter方法，允许用户访问和修改这些属性。

Venue类是一个简单的类，它代表一个场地，如剧院、体育场或竞技场。它具有名称和位置属性以及相应的get和set方法。它打算用作其他类（如Theater）可以扩展的基类，以继承其属性和方法。
